Dúvidas sobre Scripts

O que é um Script?

Um script é um conjunto de instruções para que alguma função seja executada em algum lugar.

Como executo um Script?

Na nossa plataforma você pode ver os scripts que estão sendo criados para fazer alguma ação. Mas antes de executar algum script de sua escolha, é necessário efetuar o seu cadastro na plataforma. Se você já tiver feito o cadastro basta fazer o login.

Para acessá-los siga estes passos:

1. Vá até a barra de navegação
2. Clique no botão Login
3. Insira seu usuário e senha
4. Clique em "Entrar"
5. Vá até a barra de navegação
6. Clique em "Explorar"
7. Selecione "Scripts"
8. Escolha uma categoria
9. Escolha um script
10. Clique em "Adicionar a Minha Biblioteca"
11. Vá até a barra de navegação
12. Clique em "Minha Biblioteca"
13. Procure o script que você adicionou
14. Clique em "Executar"

Dúvidas sobre Pedidos

Como faço um pedido?

Para fazer um pedido é necessário estar logado na plataforma.

Para acessá-los siga estes passos:

1. Vá até a barra de navegação
2. Clique no botão “Login”
3. Insira seu usuário e senha
4. Clique em "Entrar"
5. Vá até a barra de navegação
6. Clique em “Pedidos”
7. Selecione “Fazer um pedido”
8. Coloque as informações referentes ao script que você quer

Dúvidas sobre a plataforma

Como faço o cadastro na plataforma?

Para fazer um cadastro siga estes passos:

1. Vá até a barra de navegação
2. Clique no botão “Cadastrar”
3. Insira os dados nos campos de texto
4. Clique em “Cadastrar”
